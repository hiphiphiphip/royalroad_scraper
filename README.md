Just Good Enough Royal Road Scraper                                                                                
===================================                                                                                
                                                                                                                   
Scrapes fics from royalroad.com and dumps them in a text file.                                                     
                                                                                                                   
Install with:                                                                                                      
    pip3 install requests BeautifulSoup4 blindspin                                                                 
                                                                                                                   
Run:                                                                                                               
    python3 royalroad_scraper.py <url> <file>
